MMBASIC scientific calculator for the PicoCalc hardware that solves
math expressions as they would normally be written (not RPN). For example:
    8^2+6 <enter> would result in 70.
Each expression result is assigned to a rolling variable A thru F. 
Those variable letters may be used in subsequent calculations. 
Three additional variables X, Y & Z may be user assigned.
PicoCalc Calculator uses the MMBASIC Eval function to 
"calculate" results.

PicoCalc with Pico2W (RP2350A) v6.02.00a
